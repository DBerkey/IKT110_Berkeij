How to Use

  1. Select the hole you used for measurement.
  2. Choose how much the object covered the hole (Full, Three-Quarter, Half, Quarter).
  3. Click Calculate to get the estimated distance.
  4. For advanced options, click Advanced Options to:
    • Set the object size (in meters).
    • Set the distance from the hole to your eye (in centimeters).
    • Enable Intergalactic Measurement to get the most accurate results.

Notes

  • All calculations are estimates and may vary based on real-world conditions or the observer's skill level.
  • Object size should be entered in meters, hole-to-eye distance in centimeters.
  • Coverage modifier helps adjust for partial coverage of the hole by the object.
